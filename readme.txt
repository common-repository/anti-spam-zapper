Anti-Spam Zapper will automatically remove spam comments and block spambots without annoying CAPTCHAs. 

Protect your website from spam comments and automated spam using this free WordPress anti-spam plugin.

**Features**

- Protect Comments
- Block Spambots
- Marks comments as spam, without deleting them, just-in-case there's a false positive
- Nothing to fill out, completely silent action


**FAQ**

- Does this require javascript?
- Yes, however the vast majority of users have javascript enabled (over 99%). Users with javascript disabled will be shown a message, just to make sure nothing is lost.
